# 庐山行程网页使用说明

## 本地查看

这套网页是纯静态文件，不依赖 Node、pnpm 或其他构建工具。

直接打开下面这个文件即可：

- `index.html`

如果浏览器限制本地文件行为，也可以在当前目录执行：

```bash
python3 -m http.server 8080
```

然后访问：

- `http://localhost:8080`

## 当前目录情况

当前目录 `不是 git 仓库`，也 `没有现成的 Gitee 远端和账号授权信息`，所以我没有办法直接替你读取 Gitee 账号、仓库名或自动发布。

## 发布到 Gitee Pages 的做法

1. 在 Gitee 新建一个仓库。
2. 把下面这些文件上传到仓库根目录：
   - `index.html`
   - `styles.css`
   - `app.js`
3. 如果你也想保留原始文档，可以把这几个 `md` 文件一起传上去：
   - `20260313-0315-庐山行程规划.md`
   - `20260313-0315-庐山行程-微信群公告版.md`
   - `20260313-0315-庐山行程-按小时详细版.md`
   - `20260313-0315-庐山行程-预算与打包清单.md`
4. 在 Gitee 仓库设置里开启 `Gitee Pages`。
5. 选择部署分支，通常是 `master` 或 `main`。
6. 部署完成后，就可以直接通过 Gitee Pages 链接访问。
